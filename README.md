# Barndoor AI (barndoor)
Barndoor AI is the control plane for agentic AI, providing secure access and governance for AI agents and Model Context Protocol (MCP) servers. Founded in 2024 by Oren Michels (founder of Mashery), Barndoor enables enterprise teams to orchestrate AI agent workflows and maintain security across distributed AI systems.
