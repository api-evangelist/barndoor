# Barndoor AI (barndoor)
Barndoor AI is the control plane for agentic AI, providing secure access and governance for AI agents and Model Context Protocol (MCP) servers. Founded in 2024 by Oren Michels (founder of Mashery), Barndoor enables enterprise teams to orchestrate AI agent workflows, govern MCP server access, and maintain security and accountability across distributed AI systems. The platform supports SaaS, private cloud, and on-premises deployments with Python and TypeScript SDKs.

## Features

| Name | Description |
|------|-------------|
| MCP Governance | Secure access control and governance for Model Context Protocol servers. |
| Runtime Policy Enforcement | Continuous governance applied at the moment AI agents act. |
| AI Agent Orchestration | Control plane for managing and coordinating AI agent workflows. |
| SDK Integration | Python and TypeScript SDKs for connecting to governed MCP servers. |
| Enterprise Security | Data governance, access policies, and accountability for AI deployments. |
| Platform Connectors | Pre-built MCP catalog with Salesforce, Notion, GitHub, and more. |
| Audit and Compliance | Complete audit trails for AI agent actions and policy decisions. |
| Shadow AI Discovery | Centralized visibility into unauthorized AI apps and agents. |
